    Alexa LED Alarm System

    This project uses C language and an ESP8266 based system in order to moderate the control of a custom LED strip. With the help of third party applications and libraries I was able to connect this wireless board using WIFI to a remote server which handled the requests coming from Alexa within my room. I then coded functions like fade-in and fade-out wake up cycles and also brightness adjustment. Unfortunately the project failed later on due to overheating.
